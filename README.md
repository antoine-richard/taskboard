The "no HTML5 yet" Taskboard
============================

Ce dépôt abrite l'application servant de support au concours HTML5 Nantes.

Deux branches sont disponibles :
* La branche taskboard-maven porte l'application au format Maven
* La branche taskboard-eclipse porte l'application dans un projet Eclipse

Basculez sur celle qui vous convient le mieux.

Utilisation de l'application Taskboard avec Maven
=================================================

Pré-requis
---------
* JDK >= 6
* Maven 3
* MongoDB
